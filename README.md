Project Overview:
       This Project predicts whether a student will pass or fail 
       based on study hours and student marks using machine learning.

Problem Statement:
        Predict the result(pass/fail)of a student using:
          *study hours
          *student marks
Dataset features:
        *study_hours
        *student_marks
Target veriable:
         *Result(pass/fail)
Machine Learning Algorithm:
          *LogisticRegression
Libraries used:
        *pandas
        *numpy
        *scikit learn
        *streamlit
        *joblib
Project files:
        *day15_1.py->model training and evaluation
        *app.py->streamlit web application
        *README.md->project documentation
